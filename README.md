# 🚗 Car Sales Data Analytics Dashboard | Power BI

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze and visualize car sales performance. By leveraging Exploratory Data Analysis (EDA) and robust data modeling techniques, this report transforms raw sales data into actionable business intelligence. 
The dashboard provides stakeholders with a clear view of market trends, sales efficiency, and consumer purchasing behaviors, drawing on domain expertise in automotive sales and vehicle financing to highlight the metrics that matter most to the bottom line.

## 🎯 Business Context & Objectives
In the competitive automotive market, understanding the nuances of sales velocity, customer preferences, and financing attach rates is critical. The primary objectives of this report are to:
**Track Key Performance Indicators (KPIs):** Monitor total revenue, units sold, and average transaction values.
**Identify Market Trends:** Analyze sales distributions across different vehicle makes, models, and body types.
**Optimize Sales Strategies:** Uncover patterns in seasonal buying and regional performance to guide future inventory and sales targeting.

## 🛠️ Technical Stack & Methodologies
**Business Intelligence Tool:** Microsoft Power BI (`.pbix`)
**Data Processing:** Power Query for data cleaning, transformation, and shaping.
**Analytical Techniques:** Exploratory Data Analysis (EDA) to detect outliers, handle missing values, and establish relationships within the dataset.
**Calculations:** Advanced DAX (Data Analysis Expressions) for creating custom measures, rolling averages, and dynamic KPIs.
**Data Modeling:** Built a normalized relational data model (Star Schema) to ensure optimal filtering and performance.
